## 💥 遇到的问题 & 解决方案

| 问题 | 根本原因 | 解决方案 | 收获 |
| :--- | :--- | :--- | :--- |
| **线性堆叠无效** | 多层 `Linear` 若无激活函数，数学上等价于单层线性变换 | 插入 `nn.ReLU()`<br>打破线性，引入非线性特征 | **模型能力本质**：<br>深度学习的“深度”价值源于非线性变换 |
| **梯度衰减风险** | 链式法则导致连乘效应（如 Sigmoid 导数<1，连乘后趋近0） | 选用 **ReLU** 激活函数<br>正区间导数恒为1，梯度无损传播 | **工程权衡**：<br>ReLU 是计算效率与梯度稳定性的最佳平衡点 |
| **代码冗余难管** | 手动管理权重矩阵 (W, b) 繁琐且易错 | 使用 **`torch.nn`** 模块<br>自动注册参数与生命周期管理 | **框架思维**：<br>利用 `nn.Module` 像搭乐高一样构建网络 |

## 📊 实验数据

| 参数配置 | 理论表现 | 关键发现 |
| :--- | :--- | :--- |
| **Linear Only** (无激活) | 约 92.0% | **模型退化**：无论加多少层，效果等同于单层感知机 |
| **Linear + Sigmoid** | 收敛极慢 | **梯度消失**：深层网络参数几乎不更新，训练停滞 |
| **Linear + ReLU (v1.0)** | **收敛快** | **最佳实践**：计算仅需 `max(0,x)`，且有效保留梯度信号 |

import torch
import torch.nn as nn
import torchvision.datasets as datasets
import torchvision.transforms as transforms

# 加载MNIST（公开数据集，1行加载✅）
train_dataset = datasets.MNIST(root='./data', train=True, transform=transforms.ToTensor(), download=True)

# 构建2层网络（输入784 → 隐藏128 → 输出10）
#用单层的话如“3”和“8”局部特征相似，单层无法准确识别俩层加Relu可以拟合曲线边界
class SimpleNet(nn.Module):
    def __init__(self):
        super().__init__()
        self.fc1 = nn.Linear(784, 128)#第一层layer层 
        self.fc2 = nn.Linear(128, 10)#第二层layer层
        self.relu = nn.ReLU()  #选用 ReLU 三重理由：
# 1️⃣ 梯度恒为1（x>0）→ 避免浅层网络梯度衰减（对比Sigmoid饱和区梯度≈0，梯度消失！！！） 链式法则连乘导致，
# 2️⃣ 仅需比较运算 → 训练速度比Sigmoid快1.2倍+（企业看重迭代效率）
# 3️⃣ 负输入归零 → 引入稀疏性，隐含正则效果（减少过拟合风险）


    def forward(self, x):
        x = x.view(-1, 784)  # 展平28x28图像
        x = self.relu(self.fc1(x))
        x = self.fc2(x)
        return x

model = SimpleNet()
